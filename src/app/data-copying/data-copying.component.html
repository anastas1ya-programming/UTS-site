Копирование данных в Форму УТС из других источников:

Копирование данных в Форму УТС из других источников допускается, но при этом нужно соблюдать ряд рекомендаций:

  1. выбирать левую-верхнюю ячейку для вставки данных нужно ПРАВОЙ кнопкой мыши.
  2. В открывшемся меню необходимо найти «Параметры вставки» и кликнуть по значку «Папка с цифрами 123». При этом в Форму УТС будут скопированы только значения данных и не будет нарушено форматирование самой Формы;
  3. Если в открывшемся меню не будет «Папки с цифрами 123», то кликаем по значку «Папка с пятью синими полосками» - «Использовать конечное форматирование»:
  4. после вставки массива данных в Форму УТС, необходимо выделить часть данных содержащую Гос.номера и очисть её от пробелов и тире. Затем выделить часть данных содержащую числовую информацию и очисть её от пробелов. Для этого, после выделения соответствующей части данных, нужно:
    1. 1. нажать сочетание клавиш Ctrl+Р (русская буква «Р»);
    2. 2. ввести в поле «Найти:» один пробел;
    3. 3. поле «Заменить на:» оставить пустым;
    4. 4. кликнуть по «Заменить все».
  5. В случае с Гос.номерами, повторить данную операцию для символа «тире».
